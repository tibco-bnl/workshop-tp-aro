# Release Notes - v1.14.0

**Release Date:** February 27, 2026  
**Last Updated:** March 2, 2026  
**TIBCO Platform Control Plane Version:** 1.14.0

## Overview

This release contains documentation and scripts aligned with **TIBCO Platform Control Plane version 1.14.0** for deploying TIBCO Platform on Azure Red Hat OpenShift (ARO) clusters.

## Update Log

### March 2, 2026
- ✨ **Added GitHub Pages support** for proper Mermaid diagram rendering
  - Added `_config.yml` with Jekyll configuration
  - Added `index.md` as GitHub Pages landing page
  - Added YAML front matter to all howto documentation files
  - Enabled `parse_block_html` in kramdown for Mermaid diagrams
  - Added custom Jekyll layout (`_layouts/default.html`) with Mermaid.js CDN integration
  - Added JavaScript to convert markdown Mermaid blocks to rendered diagrams
  - Enabled `auto_ids` in kramdown for heading anchor links
  - Added custom CSS for better diagram, table, and content styling
  - **Fixed:** Architecture diagrams now display correctly on GitHub Pages
  - **Fixed:** Heading anchor links now work properly

## What's Included

### Documentation
- Complete setup guides for ARO cluster deployment with Control Plane and Data Plane
- Instructions for hybrid deployments (SaaS Control Plane with ARO Data Plane)
- Observability setup guides with Prometheus and Elastic Stack (ECK)
- DNS configuration and certificate management guides
- Security Context Constraints configuration for OpenShift

### Scripts
- ARO cluster creation automation
- PostgreSQL and MailDev setup scripts
- Certificate generation and management
- DNS configuration automation
- Environment variable configurations

## Key Features

- ✅ Full ARO cluster setup with TIBCO Platform CP + DP
- ✅ Hybrid cloud deployment support (SaaS CP + ARO DP)
- ✅ Comprehensive observability with Prometheus and ECK
- ✅ Wildcard DNS and certificate management
- ✅ OpenShift-specific security configurations
- ✅ Multi-cluster deployment support
